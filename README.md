# Smart Mess Management System

This project helps in managing hostel mess efficiently using QR-based attendance, demand prediction, and waste analysis.

## Features
- QR-based attendance system
- Food demand prediction using ML
- Waste tracking and analysis
- Stock and menu optimization

## Tech Stack
- Frontend: React
- Backend: Node.js / Flask
- Database: MySQL
- ML: Linear Regression

## How to Run
1. Start frontent and backend server by using "npm start"
2. Inside ml_service , run "python ml_app.py"
3. Import database using smart_mess.sql
